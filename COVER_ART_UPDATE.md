# Обновление: Обложка в меню выбора трека

## Новая функциональность
Добавлено отображение обложки трека в меню выбора действия (скачать трек/клип).

## Что изменилось

### Новая функция `_send_track_selection_with_cover()`
- Отправляет сообщение с обложкой трека
- Если обложка недоступна, использует текстовое сообщение
- Автоматически конвертирует `cover_uri` в полный URL
- Включает обработку ошибок с fallback

### Обновленный код в callback handler
- Заменен `bot.send_message()` на `_send_track_selection_with_cover()`
- Сохранена вся существующая функциональность кнопок

## Как это работает

1. **Пользователь нажимает на трек**
2. **Бот получает информацию о треке** (включая обложку)
3. **Пытается отправить фото с обложкой** и кнопками действий
4. **Если обложка недоступна** - отправляет текстовое сообщение
5. **Кнопки действий работают как раньше**

## Преимущества
- ✅ Визуально более привлекательный интерфейс
- ✅ Мгновенное узнавание трека по обложке
- ✅ Сохранение обратной совместимости
- ✅ Обработка ошибок без падения бота

## Технические детали
- Использует `bot.send_photo()` для изображений
- Fallback на `bot.send_message()` при ошибках
- Поддерживает как полные URL, так и относительные `cover_uri`
- Логирование ошибок для отладки

## Применение
Эта функция используется в:
- Выборе трека из поиска
- Выборе трека из чартов
- Выборе трека из избранного
- Выборе трека из альбомов исполнителя
